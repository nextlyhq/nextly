# @revnixhq/storage-vercel-blob

[Vercel Blob](https://vercel.com/docs/storage/vercel-blob) storage adapter for [Nextly](https://github.com/revnix/nextly-dev).

## Installation

```bash
pnpm add @revnixhq/storage-vercel-blob
```

`@revnixhq/storage-vercel-blob` is a peer dependency of `@revnixhq/nextly`. Install both side by side and configure in your Nextly config. Requires the `BLOB_READ_WRITE_TOKEN` environment variable from your Vercel project.
