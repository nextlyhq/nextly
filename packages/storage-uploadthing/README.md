# @revnixhq/storage-uploadthing

[UploadThing](https://uploadthing.com) storage adapter for [Nextly](https://github.com/revnix/nextly-dev).

## Installation

```bash
pnpm add @revnixhq/storage-uploadthing
```

`@revnixhq/storage-uploadthing` is a peer dependency of `@revnixhq/nextly`. Install both side by side and configure in your Nextly config.
