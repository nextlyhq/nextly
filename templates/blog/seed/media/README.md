# Seed Media Files

This directory contains sample images for the blog template's demo content.

## Required files

Post featured images (1200x630, JPEG, ~80KB each):

- `post-nextly-cms.jpg`
- `post-server-components.jpg`
- `post-typescript-tips.jpg`
- `post-tailwind-guide.jpg`
- `post-open-source.jpg`

Author avatars (400x400, JPEG, ~30KB each):

- `author-jane.jpg`
- `author-john.jpg`

Site logo (PNG, ~10KB):

- `logo.png`

## Sourcing images

Use royalty-free images from Unsplash (https://unsplash.com) or Pexels (https://pexels.com).
Choose tech/coding themed images for posts and professional headshots for avatars.
Compress images to web sizes before placing them here.

## If images are missing

The seed script handles missing images gracefully. Posts and authors will be
created without images, and a warning will be logged. You can add images later
through the admin panel.
