# Blank template

The minimum viable Nextly project. Empty `nextly.config.ts`, no collections, no frontend pages beyond what the `base` template provides.

## When to use

Choose `blank` when you want to build everything from scratch. Pick [`blog`](../blog/README.md) instead if you want a populated example to read and modify.

## Scaffold

```bash
pnpm create-nextly-app@latest --template blank my-app
```

See [`templates/blog/README.md`](../blog/README.md) for an example of how a populated template is structured.
